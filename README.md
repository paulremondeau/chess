# A self-made chess game
This repo contains a self-made chess game in Scala

# How to run locally

## 1. Backend

Tested for Scala 3.3.0

In the backend folder, run
```bash
$ sbt run
```

## 2. Frontend

In the frontend folder, run

```bash
$ npm run build
```
```bash
$ npm run preview
```
